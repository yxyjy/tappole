# Tappole: Senior-Centric Video Communication Platform

![Flutter](https://img.shields.io/badge/Flutter-3.x-blue?logo=flutter)
![Supabase](https://img.shields.io/badge/Supabase-Backend-green?logo=supabase)
![ZegoCloud](https://img.shields.io/badge/ZegoCloud-RTC-orange)
![License](https://img.shields.io/badge/License-MIT-purple)

**Tappole** is a cross-platform mobile application designed to bridge the digital divide for the elderly. It facilitates accessible, one-tap video connections between seniors and verified volunteers, ensuring social inclusion through a safe, simplified, and high-contrast interface.

This project was developed as a **Final Year Project (FYP)** to address the technical and usability barriers faced by the aging population in digital communication.

* **Frontend:** [Flutter](https://flutter.dev/) (Dart) - Chosen for its high-performance Impeller rendering engine and cross-platform capabilities.
* **Backend:** [Supabase](https://supabase.com/) - Utilized for Authentication and PostgreSQL database with Row Level Security (RLS).
* **Real-Time Communication (RTC):** [ZegoCloud UIKit](https://www.zegocloud.com/) - Handles network switching and video streaming logic.
* **Architecture:** Layer-First Architecture (Separation of UI, Logic, and Data).
